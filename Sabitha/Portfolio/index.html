Projects

Project 1

My first project, the Netflix Clone, is a web application designed to replicate the core functionalities and user interface of the popular streaming service, Netflix. This clone includes essential features such as user authentication, a responsive design, browsing and searching for movies and TV shows, personalized recommendations, and video playback. The application allows users to create accounts, log in, and manage their watchlists, simulating a real-world streaming experience.

Project 2

My second project, the Online IDE, is a web-based application that allows users to write, edit, compile, and run code directly from their web browsers. This IDE supports multiple programming languages, providing a versatile platform for developers, students, and hobbyists to work on their coding projects without the need to install software on their local machines. Key features include syntax highlighting, code completion, real-time collaboration, version control integration, and debugging tools. The IDE aims to streamline the coding process by offering a seamless and accessible environment for code development and testing.
